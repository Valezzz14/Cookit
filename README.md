# Cookit
ofrecer recetas saludables, fáciles de preparar y adaptadas a diferentes necesidades alimenticias (vegetarianas, veganas, sin gluten, bajas en calorías, etc.)
ste sitio web estará alineado con el bienestar y la salud nutricional. Su objetivo principal es ofrecer recetas saludables, fáciles de preparar y adaptadas a diferentes necesidades alimenticias (vegetarianas, veganas, sin gluten, bajas en calorías, etc.). Además, busca fomentar hábitos alimenticios positivos y educar a los usuarios sobre los beneficios de una buena nutrición.
